# web-dev-lab

Web front end development project starter


## Usage

To initialize a new project with web-dev-lab:

* `npm install -g gnat`
* `gnat clone https://github.com/reykjavikingur/web-dev-lab.git <<YOUR_DIRECTORY>>`
* `cd <<YOUR_DIRECTORY>>`
* `npm install`
* `npm start`
